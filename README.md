# Ruby_puro

Dicas valiosas:

O `%w` Ajuda para não precisarmos escrever tudo com aspas e vígula.

Bad Code:   a = ["ant", "bee", "cat", "dog", "elk"]   #saída de a => ["ant", "bee", "cat", "dog", "elk"] 

Good Code:  a = %w[ant bee cat dog elk]               #saída de a => ["ant", "bee", "cat", "dog", "elk"]
